# Swiggy Sales Dashboard (Excel)

## 📌 Project Overview
This project is an Excel-based interactive dashboard designed to analyze Swiggy-style food delivery sales data. It visualizes key performance metrics such as sales trends, ratings, orders, and regional insights in a clear and structured format.

This dashboard was created as a learning project to practice Excel data cleaning, KPI tracking, and dashboard design.

---

## 🎯 Objectives
- Organize raw sales data for analysis  
- Track KPIs like sales, ratings, and order trends  
- Visualize performance by time and region  
- Improve reporting clarity through dashboards  

---

## 🛠 Tools & Skills Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Visualizations  
- Slicers & Filters  
- Basic Excel formulas  

---

## ⚙ Dashboard Features
- Monthly, weekly, and daily sales trend analysis  
- KPI summary section  
- Region-based performance insights  
- Interactive filtering using slicers  
- Clean and visual dashboard layout  

---

## 📊 Outcome
The dashboard converts raw sales data into meaningful visual insights, making KPI tracking faster and easier. It demonstrates practical Excel dashboard-building skills.
